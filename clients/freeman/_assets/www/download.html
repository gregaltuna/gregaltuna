Android Installation Instructions

Step 1
    Prep your Device - To ensure your device can download the beta version app file, simply go to:
  • Security in the settings menu and check the box marked “UNKNOWN SOURCES“.
    (A warning message will be displayed; click OK)
Step 2

Download the Freeman App File   Click here to Download

Step 3
    Google Play Store - Search for “App Installer” (By Fun Trigger)
  • Click Install
Step 4
    Access the Freeman App
  • Open the App Installer.
  • Scroll to find Download or Root directory
    • Select the FOL Mobile file.
    • Click the Install button.
    • After installation, select the FOL MOBILE APP from the apps folder and tap to run the application.
